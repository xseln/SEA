# SH Beauty Salon

##  Цел
Уеб приложение за управление на записвания в салон за красота.

##  Технологии
- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- HTML, CSS, JavaScript

##  Основни функции
- Регистрация и вход
- Запазване на час
- Редакция на резервации
- Клиентски профил
- Административен панел

##  Стартиране
1. Отвори проекта във Visual Studio
2. Изпълни Update-Database
3. Стартирай приложението

##  Употреба
Потребителят избира услуга, служител и час и прави резервация.
